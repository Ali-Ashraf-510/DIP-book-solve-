## إعداد البيئة (conda)

ملف `environment.yml` موجود في جذر المشروع. لإنشاء نفس البيئة باسم `collage` ثم تفعيلها:

```powershell
conda env create -n collage -f environment.yml
conda activate collage
```

لو البيئة موجودة وتريد تحديثها:

```powershell
conda env update -n collage -f environment.yml
```
